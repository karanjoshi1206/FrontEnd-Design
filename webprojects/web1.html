Sorting Visualizer

Riyaz AhmadDec 1, 2020

Developed primarily using Vanilla JavaScript, this tool is used to visualise how different sorting algorithms work . Select your favourite algorithm, set your preferred size of the array and sorting speed by the sliders provided, hit sort and see the algorithm work. Execution time shows how much time your algo took in sorting all values relative to the speed you set.

Color Indicators: 1.Yellow: Element one which the algorithm is working on at a given instant. 2.Red: Elements which the algorithm wants to update. 3.Green: Elements that are sorted(i.e are in their final position in the sorted array).

Algorithms implemented:
  • 1.Quick sort
  • 2.Heap sort
  • 3.Merge sort
  • 4.Radix sort
  • 5.Selection sort
  • 6.Bubble sort
