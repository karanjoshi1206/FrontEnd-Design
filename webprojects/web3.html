News App

Karan JoshiNov 5, 2020

NewsHub is a collection of news from all over the world . To get the lastest news enter Category(like sports,politics,music etc) and your alpha country code (like in for INDIA , br for BRAZIL) and click on Get News. It take the news category and country ode as the user input and then append the input in the API (Gnews) which returns the data in the JSON format which is then converted into array using JavaScript. Then it update the front end of the website and display the News
