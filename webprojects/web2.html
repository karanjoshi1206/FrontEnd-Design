Weather App

Karan JoshiNov 5, 2020

WeatherKnow is application through which you can check the weather of any city with just one click. It take the name of the place as the user input and then append the input in the API (openweathermap.org) which returns the data in the JSON format which is then converted into array using JavaScript. Then it update the front end of the website and display the weather condition
